VS2022 上的 Resharper 安装

MIOOR小于 1 分钟教程ReSharper

  1. 在 Other Versions - ReSharper 上下载2022.x到2024.x之间的版本，推荐下载最新的。本教程写时最新的为2024.2.

  2. 安装或更新 ReSharper

    安装之前请先关闭所有正在运行的 VS 。之前安装过，就直接选择更新就行。没安装的为 Install 选项。ReSharper 是 C# 的辅助开发工具，ReSharper C++ 是 C++ 的辅助开发工具，推荐两个都安装。

  3. 下载破解工具并解压

    以管理员身份运行 Patcher.exe .

  4. 安装成功

    打开 VS 之后，依次选择 扩展 -> ReSharper -> Help -> License Information...，弹出窗口与下图的 License information 一致即可。
